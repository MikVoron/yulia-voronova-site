# Как добавить новые посты на страницу блога

## Шаг 1: Узнай номера постов

1. Открой свой канал в браузере: https://t.me/s/voronova_nutrition
2. Найди посты, которые хочешь показать на сайте
3. Кликни на дату поста, чтобы открыть его отдельно
4. В адресной строке увидишь ссылку типа: `https://t.me/voronova_nutrition/268`
5. Запомни число в конце (например, **268**) — это номер поста

## Шаг 2: Обнови blog.html

1. Открой файл `blog.html`
2. Найди секцию с постами (строки ~100-180)
3. Каждый пост выглядит так:

```html
<div class="telegram-post-widget">
    <script async src="https://telegram.org/js/telegram-widget.js?22" 
        data-telegram-post="voronova_nutrition/268" 
        data-width="100%"
        data-userpic="true"
        data-color="8B7355"
        data-dark-color="A89070">
    </script>
</div>
```

4. **Замени номер** `268` на номер нового поста
5. Если хочешь добавить больше постов — скопируй весь блок `<div class="telegram-post-widget">...</div>` и вставь ниже

## Пример: Добавление нового поста №275

**Было:**
```html
<div class="telegram-post-widget">
    <script async src="https://telegram.org/js/telegram-widget.js?22" 
        data-telegram-post="voronova_nutrition/268" 
        ...>
    </script>
</div>
```

**Стало:**
```html
<div class="telegram-post-widget">
    <script async src="https://telegram.org/js/telegram-widget.js?22" 
        data-telegram-post="voronova_nutrition/275" 
        ...>
    </script>
</div>
```

## Шаг 3: Сохрани и проверь

1. Сохрани файл `blog.html`
2. Обнови страницу в браузере (Ctrl+F5 или Cmd+Shift+R)
3. Новый пост должен появиться на странице

## Советы

- **Порядок постов**: Чтобы новые посты были вверху, размещай их первыми в списке
- **Количество**: Сейчас на странице 5 постов, но можно сделать больше или меньше
- **Обновление**: При публикации нового поста в Telegram нужно вручную обновить номера в blog.html

## Автоматическое обновление (продвинутый уровень)

Для автоматического подтягивания постов нужен серверный код (например, Netlify Functions). Если интересно — могу помочь настроить.
